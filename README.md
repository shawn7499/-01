# Wick Portfolio

深色极简 Web3 风格个人网站

## 特性

- 🌑 深色主题
- ⚡ 极简设计
- 🔷 Web3 美学（霓虹效果、玻璃态）
- 🌐 中英文切换
- 📱 响应式设计
- ✨ 流畅动画

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000

## 部署到 Vercel

1. 推送代码到 GitHub
2. 在 Vercel 导入项目
3. 自动部署完成

## 自定义内容

编辑 `app/page.tsx` 中的 `content` 对象来修改网站内容。
